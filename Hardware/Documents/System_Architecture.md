# ARI v1.0 System Architecture

## Controller
- ESP32-S3

## Display
- 2.0" IPS LCD (ST7789)

## Storage
- MicroSD Card

## Time
- DS3231 RTC

## Audio
- I2S Microphone
- I2S Speaker Amplifier

## Power
- 3.7V Li-ion Battery
- Charging Module
- Power Switch

## Communication
- Wi-Fi
- Bluetooth

## Future
- Camera
- Touch Sensor
- Servo Motors
