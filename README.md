# ARI 🤖

ARI (Artificial Responsive Intelligence)

A friendly AI companion robot designed to help people with knowledge, conversation, and emotional support.

## Vision
To create an affordable AI companion that is kind, patient, and always ready to help.

## Hardware
- ESP32-S3
- IPS Display
- RTC Module
- MicroSD Card
- Speaker
- Microphone
- Rechargeable Battery
