# 🤖 ARI (Artificial Robotic Intelligence)

## Welcome

ARI is a friendly personal AI assistant robot designed to help people in their daily lives.

The goal of ARI is not to replace humans, but to become a calm, helpful, and trustworthy companion.

---

## Version

Current Version: **ARI v1.0**

Status: **Under Development**

---

## Main Features

- Voice Conversation
- Animated Face Display
- Time & Date (RTC)
- Daily Reminders
- Memory System
- Wi-Fi Connectivity
- AI Assistant
- Friendly Personality

---

## Hardware

- ESP32-S3
- 2.0" IPS TFT Display
- RTC Module
- Speaker
- MicroSD Card
- Battery
